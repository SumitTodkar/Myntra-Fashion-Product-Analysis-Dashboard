# Myntra Fashion Product Analysis Dashboard

## 📌 Project Overview

The **Myntra Fashion Product Analysis Dashboard** is an interactive Power BI project designed to analyze fashion product data from Myntra. The dashboard provides valuable insights into product categories, brands, pricing trends, ratings, discounts, and customer preferences through dynamic visualizations and KPI metrics.

This project demonstrates data visualization, business intelligence, and analytical skills using Power BI to transform raw fashion retail data into actionable insights.

---

## 🎯 Objectives

- Analyze product distribution across categories and brands.
- Identify pricing and discount trends.
- Evaluate customer ratings and product popularity.
- Compare brand performance using key metrics.
- Enable data-driven decision-making through interactive dashboards.

---

## 📊 Dashboard Features

### 📈 Key Performance Indicators (KPIs)
- Total Products
- Average Product Price
- Average Discount Percentage
- Average Customer Rating
- Total Brands

### 🏷 Brand Analysis
- Top-performing brands
- Product count by brand
- Brand-wise pricing comparison

### 💰 Pricing Analysis
- Price distribution across categories
- Average price by brand
- Discount vs selling price analysis

### ⭐ Customer Rating Analysis
- Rating distribution
- Highest-rated products
- Rating trends across brands

### 👕 Category Analysis
- Product count by category
- Popular fashion segments
- Category-wise performance metrics

### 🎛 Interactive Filters
- Brand
- Category
- Rating
- Price Range
- Discount Percentage
